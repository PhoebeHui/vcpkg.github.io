What is vcpkg?

vcpkg is a C/C++ package manager that builds libraries from source for you and works on Windows, Linux, and macOS.

Vcpkg helps you manage C and C++ libraries on Windows, Linux and MacOS. This tool and ecosystem are constantly evolving, and we always appreciate contributions!

Why vcpkg?

Easy to use

You don´t have to look up build instructions for every dependency; just run vcpkg install <library name> and you´re good to go.

Cross-platform

Support for a variety of architectures by default, or you can define your own custom builds.

Architecture-agnostic

vcpkg runs on Windows, Linux, and macOS and builds libraries for these platforms.

High-quality catalog

vcpkg has over 1300 libraries that are routinely built against each other in our CI process. This keeps the catalog working all the time, ensuring reliability and build reproducibility.

Open source project

You can view, fork, or contribute to our source code and build recipes on GitHub.
